Dogecoin signature tool

Safety

This tool is open source and entierly executed on client side. Signing a message requires a secret phrase, it will never be sent to any server. Using this tool should be safe, however don't trust, verify. Here are some safety measures you should take:
  • Download, verify and execute source code locally
  • Withdraw your Dogecoin to a new wallet before using it
  • Enable plane mode and disable your internet connection while using it
