Dogecoin signature tool

Safety

This tool is open source and entierly executed on client side. Signing a message requires a secret phrase, it will never be sent to any server. Using this tool should be safe, however here are extra safety measures you can take :
  • Withdraw your Dogecoin to another wallet before using it
  • Enable plane mode and disable your internet connection while using it
  • Download, verify and execute source code locally
